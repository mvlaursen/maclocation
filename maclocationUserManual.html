maclocation User Manual

maclocation

maclocation is a simple Mac OS X command-line tool for for selecting a network location from among those currently configured in the Network preferences panel. It is useful if you use your Mac in more than one location (at work and at home, for example).

You must have administrative privileges to install maclocation.

Installing the maclocation tool

In a Terminal shell:
  • Create a /usr/local/bin directory if it does not exist: sudo mkdir /usr/local/bin
        The permissions, owner, and group for /usr/local/bin directory should look like this:
        drwxr-xr-x root wheel
  • Copy maclocation to /usr/local/bin: sudo cp maclocation /usr/local/bin
  • Set the SetUID bit on the installed maclocation tool: sudo chmod u+s /usr/local/bin/maclocation
        The permissions, owner, and group for /usr/local/bin/maclocation should look like this:
        -rwsr-xr-x root wheel

Using the maclocation tool

Run the maclocation tool from a Terminal shell, giving it the name of the network location you want to set. The tool recognizes any of the network location names listed in the Location pop-up menu in the Network preferences panel.

Example: /usr/local/bin/maclocation Home

Location Agent

Location Agent is a Mac application that runs the maclocation tool. A typical use, when the Mac user has separate login names for each location, is to place Location Agent in the Login Items list, so that it automatically sets the network location when the user logins in.

Installing Location Agent

  • Drop a copy of Location Agent in your Applications folder.
  • (Optional) Add Location Agent to the list of Login Items in the System Preferences panel.

Using Location Agent

  • Run Location Agent and open "Location Agent" > "Preferences". Choose a location from the "Location" pop-up menu. (Each time you choose a new location from the menu, Location Agent will make that location active.)
  • The next time Location Agent runs, it will activate the network location configured in the Location Agent Preferences. If you do configure a new location, Location Agent will automatically quit after waiting a minute.
